Other HTML Tags

<meta> Tag

Defines metadata about the HTML file, it should always be placed in the <head> tag

The <meta> tag does not display anything in the webpage but is essential in giving instructions on how the webpage is displayed or searched in search engines

examples:see code

<link> Tag

Defines a "link" between the HTML file to an external source file, such as favicons (icon tab image) and stylesheets (css)

It is normally used in the <head> tag to associate an icon or a CSS file to the HTML document

It can be used by either including a JavaScript code inside the tag or using an external file using the "src" attribute
