Semantic & Non Semantic Elements

Non-Semantic Elements

These elements does not clearly define what their "content" is about.

examples are: span, i, b, div

Class and ID Attribute

  • used to identify elements
  • can be applied to any elements
  • used to label non semantic elements
  • used in styling or modifying elements

ID

used to identify a single element in our HTML Document, this can be used to change it design or modify and retrieve the element's values.

examples:see code

Hello World

PS. Same IDs should no be used on multiple elements

Class

used to identify multiple elements in our HTML Document, particularly repetitive elements that may also be used to change its design or modify and retrieve the element's values.

examples:see code

This is comment 1

This is comment 2

PS. Classes can be used on multiple elements

Semantic Elements

Elements that clearly describes what it contains, to both the developer and the browser. These are elements which have specific tag names

It does not change any styling on the actual document. It is used to structure the website in a cleaner way.

examples are: nav, header, footer, section, article and aside
